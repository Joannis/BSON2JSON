# BSON2JSON

> Swift conversions between PlanTeam.BSON and C7.JSON

Simple two extensions to convert between BSON and JSON

```swift
let myJSON = myBSON.toJSON()
```

```swift
let myBSON = myJSON.toBSON()
```
